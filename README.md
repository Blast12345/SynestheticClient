# Client Overview

The client's job is to display the colors that are generated by the server. The Server will send colors to the Gateway which then re-broadcasts the color over the ESP-NOW network.

## Configuration

The exact configuration of the client may vary from user to user (e.g. different ESP32 boards, different LED setup). You may configure this through the `platformio.ini` file. When you have made the change, upload the code to your board.

If you have issues uploading, try different board settings. Unfortunately there is no one-size-fits-all solution (at least that I am aware of).
